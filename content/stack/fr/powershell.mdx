PowerShell est un outil/framework d'automatisation et de configuration multiplateforme (Windows, Linux et macOS) qui fonctionne bien avec vos outils existants et est optimisé pour traiter les données structurées (par exemple JSON, CSV, etc.), les API REST et les objets. des modèles. Il comprend un shell de ligne de commande, un langage de script associé et un cadre pour le traitement des applets de commande.
