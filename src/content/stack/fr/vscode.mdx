Visual Studio Code est un éditeur de code source développé par Microsoft. Il est disponible pour Windows, macOS et Linux. Il est gratuit et open source. Il est basé sur Electron, un framework permettant de développer des applications desktop en utilisant des technologies web (HTML, CSS et JavaScript).
