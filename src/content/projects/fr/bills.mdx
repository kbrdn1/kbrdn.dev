Bills App est une application de gestion de factures construite pour apprendre Vue3 et Node/Express.

## Technologies utilisées:

### Backend:

- NodeJS
- Express
- MongoDB
- Mongoose

### Frontend:

- Vue3
- Vite
- TailwindCSS
- Font Awesome

## Installation

### Backend:

#### Installer pnpm

`npm install -g pnpm` or `yarn global add pnpm`

#### Installer les dépendances

```sh
cd bill-backend-app
pnpm install
```

#### Créer un fichier .env dans le dossier bill-backend-app

```sh
# ./bill-backend-app/.env
PORT=3000
VERSION=1.0.0
HOST=127.0.0.1
PORT=3000
CORS=http://localhost:5173
MONGO_USER=<your mongo user>
MONGO_PASS=<your mongo password>
MONGO_CLUSTER=<your mongo cluster>
MONGO_DB=<your mongo db>
```

#### Lancer le serveur

```sh
pnpm start
```

### Frontend:

#### Installer les dépendances

```sh
cd bill-frontend-app
pnpm install
```

#### Lancer l'application

```sh
pnpm dev
```
